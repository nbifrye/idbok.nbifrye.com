# 第4章: アイデンティティ連携

アイデンティティ連携（フェデレーション）は、異なる組織やドメイン間でアイデンティティ情報を安全に共有・信頼するための技術とフレームワークです。
シングルサインオン（SSO）やクロスドメイン認証の基盤となります。

## この知識領域で扱うトピック

- **フェデレーションプロトコル** — SAML、OIDC、SCIMを用いた組織間のアイデンティティ連携
- **トラストフレームワーク** — eIDAS 2.0、NIST SP 800-63などの信頼基盤
- **クロスドメインアイデンティティ** — アイデンティティリンキング、アカウントフェデレーション、JITプロビジョニング
- **エンタープライズフェデレーション** — IdPプロキシ、マルチテナント、B2Bアイデンティティ
