# 第5章: アイデンティティガバナンス

アイデンティティガバナンスと管理（IGA）は、組織内のデジタルアイデンティティの作成から廃止までのライフサイクルを管理し、適切なアクセス権を維持するための体系です。
コンプライアンス要件の遵守とセキュリティリスクの低減を両立させます。

## この知識領域で扱うトピック

- **アイデンティティライフサイクル** — プロビジョニング、デプロビジョニング、Joiner-Mover-Leaverプロセス
- **アクセス管理** — RBAC、ABAC、ReBAC、ポリシーエンジン（OPA）
- **特権アクセス管理（PAM）** — 特権アカウントの管理、ジャストインタイムアクセス、シークレット管理
- **監査とコンプライアンス** — SOX、GDPR対応、アクセスレビュー、職務分離
- **ディレクトリサービス** — LDAP、Active Directory、SCIM、クラウドディレクトリ
