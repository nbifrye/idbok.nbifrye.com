# 第8章: ゼロトラストアーキテクチャ

ゼロトラストアーキテクチャは、「決して信頼せず、常に検証する」という原則に基づくセキュリティモデルです。
ネットワーク境界ではなくアイデンティティを中心に据え、すべてのアクセスを継続的に検証します。
