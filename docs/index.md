# コンテンツ

IdBoK のコンテンツは、目的別に3つのセクションに分類されています。

## セクション一覧

| セクション                  | 目的                                                                                                  |
| --------------------------- | ----------------------------------------------------------------------------------------------------- |
| [Body of Knowledge](./bok/) | 初学者がゼロからデジタルアイデンティティを学ぶための体系的な資料。第1章から順に読むことを想定している |
| [仕様解説](./spec/)         | RFC、W3C、FIDO Alliance などの個別の技術仕様を、一仕様につき1記事で解説する                           |
| [記事](./article/)          | BoK や仕様解説に収まらない個別トピック、業界の時事的な話題、実装のヒントなどを扱う                    |

## どのセクションから読むべきか

- **デジタルアイデンティティを初めて学ぶ方**: [Body of Knowledge](./bok/) を第1章から順に読み進めてください
- **特定の仕様（RFC など）を詳しく知りたい方**: [仕様解説](./spec/) から該当の仕様を探してください
- **業界の動向や実践的なヒントを知りたい方**: [記事](./article/) を参照してください
