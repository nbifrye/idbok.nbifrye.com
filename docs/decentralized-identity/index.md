# 分散型アイデンティティ

分散型アイデンティティは、個人が自身のデジタルアイデンティティを所有・管理できるようにする技術体系です。
中央集権的なアイデンティティプロバイダーに依存せず、ユーザー自身がデータの主権を持つことを目指します。

## この知識領域で扱うトピック

- **分散型識別子（DID）** — W3C標準に基づく自己管理型の識別子。ブロックチェーンなどの分散基盤上で管理される
- **検証可能な資格情報（Verifiable Credentials）** — デジタル証明書の標準形式。発行、提示、選択的開示を実現する
- **アイデンティティウォレット** — 資格情報を保管・管理し、必要に応じて提示するためのホルダーアプリケーション
- **自己主権型アイデンティティ（SSI）** — ユーザーがアイデンティティの完全な管理権を持つという思想とフレームワーク
- **ブロックチェーンとアイデンティティ** — Ethereum、Hyperledger Indyなどの分散台帳技術を用いたアイデンティティ基盤
