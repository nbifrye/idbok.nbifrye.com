# パスワードレス認証

パスワードレス認証は、従来のパスワードに代わる、より安全で使いやすい認証方法を提供します。
フィッシング耐性、ユーザー体験の向上、認証情報の漏洩リスクの低減を実現します。

## この知識領域で扱うトピック

- **FIDO2 / WebAuthn** — W3C とFIDO Allianceが策定したWeb標準。公開鍵暗号を用いたフィッシング耐性のある認証を実現する
- **パスキー** — FIDO2をベースとした同期可能な認証資格情報。プラットフォーム間でシームレスな認証体験を提供する
- **FIDO UAF / U2F** — FIDO2の前身となるレガシープロトコル
- **マジックリンクとOTP** — メールやSMSを利用した一時的な認証方法
- **生体認証** — 指紋、顔、虹彩などの生体情報を用いた認証
